# Fireworks AI (fireworks-ai)

Fireworks AI is a production-grade inference platform for open-source and proprietary generative models. The Fireworks API hosts Llama, DeepSeek, Qwen, Mixtral, Stable Diffusion, and other models with serverless pay-per-token, on-demand dedicated GPU, and batch deployment options, plus managed fine-tuning.

## APIs

| API | Description |
|---|---|
| Fireworks Chat Completions API | OpenAI-compatible chat across 100+ open-source and proprietary models. |
| Fireworks Completions API | Legacy text completions endpoint. |
| Fireworks Vision API | Vision-language understanding via chat completions. |
| Fireworks Embeddings API | Dense embeddings for retrieval/RAG. |
| Fireworks Rerank API | Cross-encoder reranking of candidate passages. |
| Fireworks Images API | Text-to-image and image-to-image generation. |
| Fireworks Audio API | Whisper transcription/translation and TTS. |
| Fireworks Batch Inference API | Async inference at 50% off serverless rates. |
| Fireworks Fine-Tuning API | Managed SFT (LoRA/full) and reinforcement fine-tuning. |
| Fireworks Files API | Upload/manage training datasets and batch files. |
| Fireworks Models API | Lists models and deployments. |
| Fireworks Deployments API | On-demand dedicated GPU deployments with autoscaling. |
| Fireworks Account API | Account, billing, and usage management. |
